# Claude Plugins

My personal collection of Claude Code plugins.

## Using this marketplace

Add this marketplace to Claude Code:

```bash
/plugin marketplace add sharkjets/claude-plugins
```

Then install:

```bash
/plugin install gamedev-mentor@sharkjets
```
