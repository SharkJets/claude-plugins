# Claude Plugins

My personal collection of Claude Code plugins.

## Using this marketplace

Add this marketplace to Claude Code:

```bash
/plugin marketplace add sharkjets/claude-plugins
```

Then install plugins:

```bash
/plugin install ai-rubric@sharkjets
```

## Plugins

- [ai-rubric](plugins/ai-rubric/README.md) - Design, implement, and review game AI systems against a principled rubric covering enemy behaviors, NPC logic, behavior trees, dynamic difficulty, and more
