# Claude Plugins

My personal collection of Claude Code plugins.

## Using this marketplace

Add this marketplace to Claude Code:

```bash
/plugin marketplace add sharkjets/claude-plugins
```

Then install plugins:

```bash
/plugin install ai-rubric@sharkjets
```

## Plugins

- [ai-rubric](plugins/ai-rubric/README.md) - Design, implement, and review game AI systems against a principled rubric covering enemy behaviors, NPC logic, behavior trees, dynamic difficulty, and more
- [player-protection](plugins/player-protection/README.md) - Design game mechanics that align player incentives with intended gameplay — push-forward systems, reward reframing, grading, soft discouragement, and more
- [ethical-engagement-design](plugins/ethical-engagement-design/README.md) - Design player engagement through quality craft — pacing, novelty, goal structure, flow state, and dark pattern avoidance
- [game-design-problem-solving](plugins/game-design-problem-solving/README.md) - Diagnose and fix game design problems — root cause analysis, lever management, creative reframing, and playtesting methodology
- [combat-design](plugins/combat-design/README.md) - Design melee combat systems that feel satisfying — attack animation phases, hit stop, game feel, defensive mechanics, style scoring, and enemy telegraphs
- [puzzle-design](plugins/puzzle-design/README.md) - Design puzzles that make players feel smart — catch and revelation structure, misdirection, minimalism, difficulty curves, and playtesting methodology
- [gamedev-mentor](plugins/gamedev-mentor/README.md) - Surface design problems you may not know you have — codebase audit or structured consultation across all design domains
